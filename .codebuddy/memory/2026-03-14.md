## 游戏太简单 - 经验系统数值重平衡

### 问题分析
怪物经验掉落过多，核心原因是三层叠加效应：
1. **XP球掉落数 = ceil(怪物xp/2)**，怪物xp只决定球数，球值由`2+level*0.5`独立计算 → 双重乘法膨胀
2. **精英怪xp×5** → 球数暴增（精英豺狼人15xp → 8球）
3. **升级需求曲线太平缓**（8*lv^1.65 + lv*4），而经验收入远超需求
4. **升级回复20%HP** + 频繁升级 = 几乎不死
