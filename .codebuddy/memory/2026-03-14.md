## Bug修复：技能特效不消失

### 问题
用战士进游戏后技能特效不消失，效果堆积。

### 根因
`processSkills()` 中的 `resetCd(k,sk,l)` 辅助函数定义错误：
```js
// 错误：自身递归调用，导致无限递归→栈溢出→CD永远不被重置→技能每帧触发
function resetCd(k,sk,l){resetCd(k,sk,l);triggerCastAnim()}
```
应该调用 `skillTimers[k]=calcSkillCd(sk,l)` 来重置CD计时器。

### 修复
```js
function resetCd(k,sk,l){skillTimers[k]=calcSkillCd(sk,l);triggerCastAnim()}
```
此修复同时影响所有14个技能+合成技能以外的所有普通技能CD重置。
