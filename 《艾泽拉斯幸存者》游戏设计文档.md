# 《艾泽拉斯幸存者 — Azeroth Survivors》

## 完整游戏设计文档 v1.0

> **品类定位**：割草 × Roguelike × 魔兽世界IP融合  
> **目标平台**：iOS / Android  
> **核心设计目标**：次日留存 ≥55%，3日留存 ≥40%，7日留存 ≥28%  
> **变现模型**：IAA（激励视频为主）+ IAP（月卡/Battle Pass/皮肤）混合

---

# 第一部分：核心玩法设计

---

## 1.1 一句话描述

> 你是一名被传送到扭曲时空的艾泽拉斯英雄，在无尽的怪物浪潮中存活，每一局都更强，每一天都解锁新的可能。

## 1.2 核心循环（30秒体验）

```
选英雄 → 进入副本(割草战斗) → 升级拾取技能(Roguelike) → BOSS战 → 结算奖励
   ↑                                                              ↓
   ←←←←←←←←←← 外围养成 / 解锁新英雄 / 推进主线 ←←←←←←←←←←←←←←←
```

### 单局流程（一局约8-15分钟）

| 阶段 | 时间 | 内容 |
|------|------|------|
| 选择阶段 | 30s | 选英雄+选天赋路线+选副本 |
| 第1波~第3波 | 0-3min | 低密度怪物，快速升级，体验"割草快感" |
| 第4波~第6波 | 3-6min | 中密度，精英怪出现，需要走位+技能配合 |
| 第7波~第9波 | 6-10min | 高密度，屏幕铺满，视觉爽感拉满 |
| BOSS战 | 10-12min | 大型BOSS（参照魔兽BOSS机制：走位/打断/躲技能） |
| 结算 | 12-15min | 经验/金币/装备碎片/赛季积分结算 |

## 1.3 魔兽元素深度融合

### 1.3.1 英雄系统（对应魔兽职业幻想）

| 职业 | 原型 | 割草定位 | 标志性技能 | 解锁方式 |
|------|------|---------|-----------|---------|
| **战士** | 武器战 | 近战AOE | 旋风斩（360°持续旋转伤害） | 初始解锁 |
| **法师** | 火法 | 远程范围 | 烈焰风暴（屏幕范围火雨） | 初始解锁 |
| **猎人** | 兽王猎 | 召唤+远程 | 多重射击+宠物群 | 第2天登录解锁 |
| **牧师** | 暗牧 | 持续伤害 | 暗言术：痛（范围DOT扩散） | 通关第3章 |
| **盗贼** | 狂徒贼 | 高爆发 | 影舞（分身割草） | 7日签到 |
| **萨满** | 增强萨 | 图腾流 | 放置图腾自动输出 | 首充奖励 |
| **死亡骑士** | 冰DK | 减速控制 | 凛冬将至（全屏冰霜） | 赛季奖励 |

| **圣骑士** | 惩戒骑 | 近战AOE+护盾 | 复仇之盾（弹射割草） | 竞技场排名奖励 |

> **设计意图**：10个职业不在首日全部开放，通过不同解锁条件制造**阶梯目标感**，每个新英雄都是一个新的留存锚点。

### 1.3.3 Roguelike技能系统（魔兽天赋树 + 割草Build）

每次升级从3个技能中选1个（可看广告刷新为4选1），技能分为：

**基础技能（白色品质）**—— 出现概率高
- 火球术：向前方发射火球
- 霜冻新星：以自身为中心释放冰环
- 雷霆一击：对周围敌人造成闪电伤害
- 治疗之泉：每10秒恢复5%血量
- 荆棘术：被攻击时反弹伤害

**进阶技能（绿色品质）**—— 需要前置
- 活体炸弹（需火球术Lv3）：火球命中后敌人爆炸
- 暴风雪（需霜冻新星Lv3）：持续降下冰雹
- 闪电链（需雷霆一击Lv3）：闪电在敌人间弹射
- 血之渴望（需治疗之泉Lv3）：击杀回血
- 自然之怒（需荆棘术Lv3）：荆棘变成追踪飞弹

**传说技能（橙色品质）**—— 单局仅出现1-2次
- 灰烬使者：召唤灰烬使者持续旋转割草
- 霜之哀伤：每次攻击有概率冰冻全屏
- 萨格拉斯之眼：天空中出现巨眼持续射线
- 达拉然坠落：整个达拉然从天而降砸向敌群
- 时光倒流：死亡后原地满血复活一次

**终极合成（需要特定组合）**
| 合成技能 | 需要条件 | 效果 |
|---------|---------|------|
| 泰坦之握 | 火球Lv5+雷霆Lv5 | 发射巨型火雷球清屏 |
| 冰火两重天 | 暴风雪+活体炸弹 | 全屏冰火交替 |
| 灵魂收割 | 血之渴望+自然之怒 | 击杀产生追踪灵魂 |

> **设计意图**：合成系统制造"发现感"和"构筑乐趣"，让玩家每局都想尝试新组合。这是Roguelike的灵魂，也是让玩家"再来一局"的核心驱动。

## 2.2 次日留存设计（目标：≥55%）

### 2.2.1 登门槛效应 — "先让你踏进来"

**原理**：先用一个极小的请求获得同意，再逐步提出更大请求。

**具体设计**：

| 触发时机 | 小请求 | 渐进请求 | 最终目标 |
|---------|-------|---------|---------|
| 首次进入 | "选一个英雄吧！"（只需点击） | "给英雄取个名字" | 产生角色认同 |
| 首局结束 | "领取通关奖励"（一键领取） | "明天来领双倍奖励" | 次日回流 |
| 首次升级 | "解锁了新技能！试试？" | "升到Lv5解锁猎人" | 养成驱动 |
| 首次死亡 | "看广告复活继续？"（免费的） | "下次可以用钻石复活" | 建立付费认知 |

### 2.2.2 蔡格尼克效应 — "未完成的事会一直惦记"

**具体设计**：

1. **未完成的关卡进度条**
   - 玩家首日通常打到第2章中期
   - 退出时显示："距离击败范克里夫还差 23%"
   - Push通知："范克里夫在嘲笑你，英雄！"

2. **未领取的宝箱**
   - 首日结束时送一个"8小时倒计时宝箱"
   - 宝箱内容预览：显示模糊的橙色装备轮廓
   - 文案："神秘装备正在锻造中，明天回来领取！"

3. **英雄碎片差1片**
   - 首日精准给到猎人碎片 9/10
   - 第二天登录送最后1片
   - 文案："猎人雷克萨还差最后一块灵魂碎片！"

### 2.2.3 预告+稀缺效应 — "明天有好东西，错过就没了"

**具体设计**：

```
┌──────────────────────────────────────────┐
│     🔥 明日限定预告 🔥                    │
│                                          │
│  [模糊的英雄剪影] + [倒计时 23:42:18]     │
│                                          │
│  "传说中的猎人即将现身艾泽拉斯..."         │
│  "仅限明日登录的勇士可免费获得！"           │
│                                          │
│  已有 12,847 名勇士预约 ← 社会认同        │
│                                          │
│  [预约提醒] [分享给好友]                   │
└──────────────────────────────────────────┘
```

### 2.2.4 每日首胜加成

- 每天第1局通关：**奖励×3倍**
- 每天第1局选技能：**必出1个紫色品质以上技能**
- 每天第1次BOSS战：**掉落保底1个装备碎片**

> 让玩家形成"每天至少打一局"的习惯。

## 2.3 三日留存设计（目标：≥40%）

### 2.3.1 阶梯目标感效应 — "每天都有新东西解锁"

**三日解锁路线图（展示给玩家看）**：

```
Day 1                    Day 2                    Day 3
  │                        │                        │
  ▼                        ▼                        ▼
┌─────────┐           ┌─────────┐           ┌─────────┐
│ 战士+法师 │──解锁──→ │  猎人    │──解锁──→ │ 天赋系统 │
│ 第1-2章  │          │ 第3章    │          │ 装备锻造 │
│ 基础技能  │          │ 进阶技能  │          │ 挑战副本 │
│ 竞技场预览 │          │ 竞技场开放 │          │ 公会系统 │
└─────────┘           └─────────┘           └─────────┘
```

**关键心理**：玩家在Day1就能看到Day2和Day3将要解锁的内容（灰色锁定状态），制造"我还有这么多东西没玩到"的期待感。

### 2.3.2 三日登录奖励链（递增价值 + 沉没成本）

| 天数 | 奖励 | 价值感知 | 心理机制 |
|------|------|---------|---------|
| Day 1 | 100钻石 + 10次抽奖券 | ★★☆ | 登门槛：先给甜头 |
| Day 2 | 猎人英雄碎片×10（直接解锁） | ★★★ | 进阶奖励：价值递增 |
| Day 3 | **自选紫色装备宝箱** + 限定头像框 | ★★★★★ | 峰值奖励：不来就亏了 |

> **损失厌恶**：Day2登录后弹窗提示 —— "明天是3日连续登录最终奖励日！放弃将重置进度"

### 2.3.3 装备锻造系统（Day2解锁）

| 装备部位 | 来源 | 锻造时间 | 效果 |
|---------|------|---------|------|
| 武器 | 副本掉落碎片 | 2小时 | 提升基础攻击 |
| 护甲 | 竞技场奖励 | 4小时 | 提升生命值 |
| 饰品 | BOSS击杀 | 8小时 | 特殊被动技能 |
| 套装 | 集齐散件 | 即时 | 强力套装效果 |

> **设计意图**：锻造需要时间，玩家放下一件装备锻造后，必须隔几小时（或次日）回来领取。利用"离线收益+等待"制造回流理由。

### 2.3.4 竞技场系统（Day2解锁）

- **异步PVP**：用AI控制其他玩家的英雄Build
- 每天5次免费挑战次数
- 排行榜每日结算奖励
- **段位系统**：
  ```
  青铜 → 白银 → 黄金 → 铂金 → 钻石 → 大师 → 传说（宗师）
  ```
- **赛季结算**（每2周一个赛季），赛季结束发放限定奖励

# 第三部分：玩家心理效应完整应用矩阵

---

## 3.1 心理效应 × 游戏系统对照表

| 心理效应 | 定义 | 在游戏中的具体应用 | 影响的留存节点 |
|---------|------|------------------|--------------|
| **登门槛效应** | 先接受小请求，更容易接受大请求 | ①新手引导先让选英雄→取名→打一局 ②先看免费广告→再买月卡 ③先加入公会→再参与公会战 | 次日 |
| **目标感效应** | 明确的目标激发动力 | ①每章清晰的通关目标 ②英雄解锁进度条 ③竞技场段位提升 | 次日/3日 |
| **阶梯目标感效应** | 目标由近及远，逐步递增 | ①Day1-7解锁路线图 ②青铜→传说段位 ③技能白→绿→紫→橙进阶 | 3日/7日 |
| **稀缺效应** | 越稀缺越想要 | ①限时活动 ②赛季限定皮肤 ③传说技能单局仅1-2次 ④每日限量钥匙 | 7日 |
| **蔡格尼克效应** | 未完成的任务记忆更深 | ①未通关的章节 ②差1片的英雄碎片 ③锻造中的装备 ④未领取的宝箱 | 次日 |
| **损失厌恶** | 损失的痛苦>获得的快乐 | ①签到中断提醒 ②赛季通行证倒计时 ③体力溢出提醒 ④段位保护赛 | 3日/7日 |
| **禀赋效应** | 拥有的东西更珍视 | ①英雄养成后不舍得放弃 ②装备强化投入 ③公会贡献排名 | 7日+ |
| **锚定效应** | 先看到的数字影响判断 | ①商城"原价648"划掉→"限时68" ②月卡日均才2元 ③钻石首充6倍 | 付费转化 |
| **社会认同** | 别人都在做，我也应该做 | ①"XX万勇士正在战斗" ②公会排行 ③好友在线提示 ④弹幕式抽奖公告 | 3日/7日 |

## 4.1 广告变现设计（IAA）

### 4.1.1 激励视频广告点位设计

> **核心原则**：广告不是惩罚，是"额外奖励"。让玩家感觉"看广告是赚到了"而不是"被迫看广告"。

| 广告点位 | 触发场景 | 奖励 | 每日上限 | 心理学原理 | 预估观看率 |
|---------|---------|------|---------|-----------|-----------|
| **复活广告** | 战斗死亡时 | 满血复活继续 | 每局1次 | 损失厌恶（已投入10分钟不想白打） | 85% |
| **双倍奖励** | 关卡结算时 | 金币/经验×2 | 每日3次 | 锚定效应（看到翻倍的数字） | 70% |
| **技能刷新** | 升级选技能时 | 刷新3个新选项+额外1个选项 | 每局2次 | 稀缺效应（可能刷出传说技能） | 55% |
| **宝箱加速** | 宝箱倒计时中 | 减少2小时等待 | 每日5次 | 即时满足 vs 延迟满足 | 60% |
| **额外体力** | 体力耗尽时 | +30体力（约1局） | 每日3次 | 登门槛（先看广告→再买体力） | 65% |
| **每日抽奖** | 主界面转盘 | 随机道具/碎片 | 每日3次（首次免费） | 赌博心理+变比率强化 | 50% |
| **补签卡** | 签到断签时 | 补签1天 | 每月1次 | 损失厌恶（不想断签到链） | 90% |
| **竞技场次数** | 次数用完时 | +1次挑战 | 每日2次 | 目标感（差一点就升段） | 45% |

### 4.1.2 广告呈现的心理学包装

**❌ 错误做法（让玩家反感）**
```
[弹窗] 观看广告获得奖励 [看广告] [关闭]
```

**✅ 正确做法（让玩家主动想看）**

**场景1：死亡复活**
```
┌──────────────────────────────────────┐
│                                      │
│    💀 英雄倒下了...                   │
│                                      │

│    💰 获得金币: 1,250                 │
│                                      │
│    距离BOSS仅剩 8%！                  │
│                                      │
│    🎬 [英雄不灭！满血再战]  ← 观看短片  │
│    💀 [含恨离场]                      │
│                                      │
│    "真正的英雄永不放弃" — 瓦里安·乌瑞恩 │
│                                      │
└──────────────────────────────────────┘
```

> **注意**："距离BOSS仅剩8%"是精心设计的 —— 系统会在玩家快打到BOSS时适当提高难度，让死亡发生在"差一点"的时候，最大化复活广告的观看意愿。

**场景2：双倍奖励**
```
┌──────────────────────────────────────┐

│                                      │

```

> **锚定效应**：先展示双倍后的数字，让玩家对"翻倍"的价值有直观感知，再告诉他只需看一个短片。

### 4.1.3 广告频率控制（防止玩家疲劳）

```
广告频控规则：
├── 单局内：最多触发3次广告机会（复活1+技能刷新2）
├── 单日内：最多观看15次激励视频
├── 连续广告间隔：至少5分钟
├── 新手保护：Day1前3局不弹任何广告
├── 付费用户：月卡用户自动获得每日双倍奖励（减少广告需求）
└── 疲劳检测：如果玩家连续跳过3次广告，降低弹出频率24小时
```

### 4.2.3 月卡系统

| 月卡类型 | 价格 | 即时奖励 | 每日奖励 | 特权 |
|---------|------|---------|---------|------|

### 4.3.1 阶梯转化模型


# 附录：核心设计语录

---

> 1. **"永远让玩家觉得'再来一局'就能突破"** —— 这是割草Roguelike的灵魂
> 
> 2. **"广告不是中断，是选择"** —— 让玩家主动选择看广告，而非被迫
> 
> 3. **"付费是捷径，不是终点"** —— 免费玩家也能体验90%的内容
> 
> 4. **"每一天登录都有新惊喜"** —— 第一周的每一天都不一样
> 
> 5. **"社交是最强的留存黏合剂"** —— Day5之后，公会替代系统成为留存主力
> 
> 6. **"损失比获得更痛苦"** —— 善用损失厌恶，但不要让玩家感到被操控
> 
> 7. **"稀缺创造欲望"** —— 但过度稀缺会导致放弃，把握度
